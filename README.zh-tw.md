N Short URL，一款不能多得的短連結系統。 使用MDUI框架，響應式設計，完美相容PC、Android、Mac、IOS、iPad等設備。

可自行設定連結過期時間，注册登入後永久保存連結。 使用必應每日一圖作為背景，漂亮時尚。

此程式碼語言為中文簡體，可能不適合您的區域。 您可根據協定著名出處翻譯後再次發佈，謝謝！
